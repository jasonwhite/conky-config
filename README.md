# Dependencies

 * Conky with Lua support.
 * Cantarell and Impact fonts.

# Installation

Clone the repository:

    $ git clone https://github.com/jasonwhite/conky-config.git ~/.config/conky

To run it:

    $ ~/.config/conky/start
